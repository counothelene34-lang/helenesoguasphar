# helenesoguasphar

Site SOGUASPHAR pour gérer les commandes, précommandes et sondages pharmacies.

## Lancer en local

```bash
npm start
```

Le site démarre par défaut sur :

```text
http://127.0.0.1:56651
```

## Données

Les réponses, campagnes, sondages et accès pharmacies sont stockés dans le dossier `data`.
Ces fichiers ne sont pas envoyés sur GitHub car ils peuvent contenir des données sensibles.
